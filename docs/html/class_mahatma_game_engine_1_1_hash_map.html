Detailed Description

template<typename TKey, typename TData, typename HashFunction = DefaultHash<TKey>, typename ComparisonFunctor = DefaultCompare<TKey>>
class MahatmaGameEngine::HashMap< TKey, TData, HashFunction, ComparisonFunctor >

Templated HashMap class uses a key, value pair of any type with a default hash function. A user-defined hash function can be passed by using the third type argument of hash function.
