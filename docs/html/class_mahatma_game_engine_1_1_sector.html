Mahatma Game Engine
A Data-driven game engine in C++ 11 with support for XML scripting, with a provision of a World, Sector and Entity heirarchy of game objects. It also includes an Event system that runs asynchronously.

Detailed Description

A Sector is another container, but it is one which does not introduce a namespace. We should be able to move data and processes between Sectors without affecting anything in the data itself.
